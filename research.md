## ZK & L2 Security

I maintain several deep-dives into the security architecture of Zero-Knowledge systems:

- **[Circom-Security](https://github.com/this-vishalsingh/Circom-Security)** — Common vulnerability patterns in Circom circuits.
- **[Cairo-Security](https://github.com/this-vishalsingh/Cairo-Security)** — Security analysis for Cairo-based STARK systems.
- **[Noir-security](https://github.com/this-vishalsingh/Noir-security)** — Threat models and remediation for Noir circuits.
- **[zkVM-Security](https://github.com/this-vishalsingh/zkVM-Security)** — Comprehensive security analysis and vulnerability patterns for Zero-Knowledge virtual machines.
- **[L2-Security](https://github.com/this-vishalsingh/L2-Security)** — Rollup security frameworks and layer-2 threat analysis.
- **[ZK-Whiteboard](https://github.com/this-vishalsingh/ZK-Whiteboard)** — Collaborative research on new ZK primitives and vulnerabilities.
